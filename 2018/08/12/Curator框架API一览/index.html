参数说明

  • connectString:服务列表，形如“ip:port,ip:port,,,ip:port”
  • sessionTimeoutMs:会话超时时间，单位毫秒，默认60000ms
  • connectionTimeoutMs:连接创建超时时间，单位为毫秒，默认15000ms
  • retryPolicy:重试策略，默认四种实现，分别为ExponentialBackoffRetry,RetryNTimes,RetryOneTime,RetryUnitElapsed
    重试策略通过RetryPolicy来让用户自定义重试策略，RetryPlicy接口定义一个方法 参数说明
  • retryCount:已经重试的次数，如果第一次重试，该参数为0
  • elapsedTimeMs:从第一次重试开始已经花费的时间，单位为毫秒
  • sleeper:用于sleep指定时间

ExponentialBackoffRetry重试策略的构造函数

参数说明

  • baseSleepTimeMs:初始sleep时间
  • maxRetries:最大重试次数
  • maxSleepMs:最大休眠时间，毫秒
    当前次休眠时间:baseSleepTimeMs * Math.max(1, random.nextInt(1 << (retryCount + 1)))，可见随着重试次数增加，计算出的sleep时间会越来越大，如果该sleep大于maxSleepMs，则使用maxSleepMs作为休眠时间

参数说明

  • context:传递的信息，可以通过后续处理函数的CuratorEvent参数中获得
